## 🚀 Cara Upload ke GitHub & Aktifkan sebagai Home Screen App

### Langkah 1 — Upload ke GitHub
1. Pergi ke [github.com](https://github.com) → **New repository**
2. Nama repo: `phonefix-pro`
3. Set sebagai **Public**
4. Upload semua fail dari folder ini (drag & drop atau guna GitHub Desktop)
5. Pastikan struktur fail seperti berikut:
```
phonefix-pro/
├── index.html
├── manifest.json
├── sw.js
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

### Langkah 2 — Aktifkan GitHub Pages
1. Dalam repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → folder: **/ (root)**
4. Klik **Save**
5. Tunggu 1-2 minit → URL app anda: `https://[username].github.io/phonefix-pro`

### Langkah 3 — Tambah ke Home Screen (Android)
1. Buka URL app dalam **Chrome**
2. Tap menu **⋮** (tiga titik) → **Add to Home screen**
3. Atau tunggu banner "Install app" muncul automatik
4. Tap **Install** → PhoneFix Pro akan muncul di home screen seperti app biasa!

### Langkah 4 — Tambah ke Home Screen (iPhone/iPad)
1. Buka URL app dalam **Safari**
2. Tap ikon **Share** (kotak dengan anak panah atas)
3. Scroll bawah → tap **Add to Home Screen**
4. Namakan **PhoneFix Pro** → tap **Add**
